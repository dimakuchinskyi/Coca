# Coca — Digital Agency Website

## Опис проекту
Це сучасний багатосторінковий сайт digital-агентства, створений на React з використанням React Router та Supabase. Сайт має адаптивний дизайн, багату візуальну складову, інтеграцію з backend-сервісом для зворотного зв'язку та зручну навігацію.
## Основний функціонал
- **Головна сторінка** — презентація сервісу, відгуки клієнтів, email-форма.
- **Про нас** — опис агентства, переваги,
- **Ціни** — тарифні плани, FAQ
- **Блог** — категорії, слайдери, прев’ю постів.
- **Детальна сторінка блогу** — повний текст статті, автор, дата, соцмережі.
- **Контакти** — форма зворотного зв’язку (Supabase)
- **Адаптивний Header і Footer** з логотипом, навігацією та контактами.

## Структура проекту
```
src/
	App.js           # Головний компонент з роутингом
	index.js         # Точка входу
	components/      # Header, Footer
	screens/         # Home, About, Pricing, Blog, DetailBlog, ContactUs
	assets/          # Зображення, іконки
	styles/          # CSS
public/            # index.html, favicon, manifest
```

## Технології
- React
- React Router
- Styled-components
- Supabase (бекенд для контактної форми)
- CSS Modules
